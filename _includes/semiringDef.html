Definition: A semiring \((R,+,*,0_R,1_R)\) is a set \(R\) equipped with two binary operations \(+\) and \(*\), and two noted elements \(0_R\), \(1_R\) such that:
  1. \((R,+)\) is a commutatiive monoid with identity element \(0_R\).
    • \(+\) is associative: \((a + b) + c = a + (b + c)\)
    • \(0_R\) is an additive identity: \(0_R + a = a = a + 0_R\)
    • \(+\) is commutative: \(a + b = b + a\)
  2. \((R,*)\) is a monoid with identity element \(1_R\)
    • \(*\) is associative: \((a * b) * c = a * (b * c)\)
    • \(1_R\) is a multiplicative identity: \(1_R * a = a = a * 1_R\)
  3. Multiplication distributes over addition
    • \(a * (b + c) = a * b + a * b\)
    • \((a + b) * c = a * c + b * c\)
  4. Multiplication by \(0_R\) annihilates \(R\).
    • \(0_R * a = 0_R = a * 0_R\)
